# Hi, I'm Emre
**Backend & Data Engineering • Document Processing • Automation Systems**

I graduated from Gazi University – Public Finance and have been working since 2018 as an Inspector at the Social Security Institution in Türkiye.

My work involves investigating workplace accidents, auditing healthcare providers, analyzing construction labor costs and preparing detailed reports. Because of the nature of the institution, these investigations often require working with large amounts of documents and complex data.

**Over time I started asking a simple question:
Can some of these processes be automated?** 

This question led me into software development. My professional background in inspection gives me a unique **"Inspector's Eye"** for data. I don't just write code; I prioritize data integrity, security and logical precision.

## What I Like Working On

- transforming unstructured files (PDF, XML, etc.) into structured databases
- designing data models for messy real-world data
- building automation tools
- creating data pipelines
- generating automated reports
- building AI-assisted workflows
- experimenting with LLM integrations (RAG, fine-tuning)
